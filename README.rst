Strawberry Limes 
================
OpenLayers-basierte Karten-Webanwendungen mit zeitbasierten POIs

Copyright\(C) 2013 Johannes Kroll/Wikimedia Deutschland e.V.

Die ursprüngliche Umsetzung der Software wurde für die `Limeskarte <https://tools.wmflabs.org/render-tests/limes/web/>`_ entwickelt. Dieses Projekt erweitert die Software und macht die Darstellung und das Datenmodell konfigurierbar, sodass die Karte auch in anderen Projekten verwendet werden kann, zum Beispiel im geplanten `Pilotprojekt zu abgebrochenen Ortschaften im Zusammenhang mit Braunkohletagebau <https://de.wikipedia.org/wiki/Wikipedia_Diskussion:Bautzen#Pilotprojekt_zu_W.C3.BCstungen_und_abgebrochenen_D.C3.B6rfern>`_.
